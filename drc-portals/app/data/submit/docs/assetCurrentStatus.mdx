### Current

An asset tagged by the <BsCheckCircleFill style={{ display: 'inline-flex', alignItems: 'center' }}/> icon under the 'Current' column on the Uploaded Assets page is considered the current version of that file type for a given DCC.
![Screenshot showing current asset](/img/submission_system/current_asset.png)
*Figure 1: A screenshot of Uploaded Assets page showing a current asset. View of Uploader User*



### Archived

An asset tagged by the  <FaCircleExclamation style={{ display: 'inline-flex', alignItems: 'center' }} /> icon under the 'Current' column on the Uploaded Assets page, is considered an archived version of that asset type. Please note that both DCC and DRC approvers can change the current status of an asset.
![Screenshot showing archived asset](/img/submission_system/archived_asset.png)
*Figure 2: A screenshot of Uploaded Assets page showing an archived asset. View of Uploader User*
